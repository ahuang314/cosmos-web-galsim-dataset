# Overview

This is the documentation page for the dataset uploaded [here](https://zenodo.org/records/19188494) on Zenodo. 

This dataset, meant for use with SLSim, is a downselected version of the full COSMOS Web dataset, of which the master catalog `COSMOSWeb_mastercatalog_v1.fits` and NIRCam images can be downloaded [here](https://cosmos2025.iap.fr/catalog.html) and [here](https://cosmos2025.iap.fr/nircam.html#downloads).

The notebook used to generate this downselected catalog from the full dataset is provided in this repository.

# Cuts Applied to the Catalog

The downselected catalog was generated by applying the following cuts to the master catalog:

- magnitude < 20 in at least one band
- warn_flag = 0 (to ensure high quality sources; see documentation about quality flags [here](https://cosmos2025.iap.fr/catalog.html#quality-flags))
- redshift != -99 (some galaxies have their redshifts in the catalog set to -99; the documentation does not explain why)
- type = 0 (to select galaxies only; type = 1 is stars and type = 2 is QSOs)
- sources with contaminants that are within 85 pixels are excluded
- sources on the edge of the detector are excluded

# Description of the dataset

The downselected catalog is stored in the file named `COSMOSWeb_galaxy_catalog.fits`. This file contains an astropy Table with the following data:

- id: The identification number of the source
- angular_size: The geometric mean of the fitted Sersic radii along the major and minor axes, in arcseconds
- sersic_index: The fitted Sersic index
- sersic_angle: The fitted Sersic angle in radians, measured clockwise from the +y axis
- axis_ratio: The fitted axis ratio
- z: redshift

The images corresponding to each galaxy are stored in a file named `COSMOSWeb_galaxy_{id}_image.fits`, where {id} is the id of the source. Inside this file is an empty PrimaryHDU followed by one ImageHDU for each band F115W, F150W, F277W, and F444W containing a 121x121 pixel cutout of the source.
